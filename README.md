
# Ter

> Ter 프로젝트 설명

나만의 홈쇼핑 채널을 만들어주는 ter project.
로그인 후, Broadcast버튼을 클릭하면 나만의 홈쇼핑 방송이 시작됩니다.
Room버튼을 클릭하면, 현재 서버에서 실시간으로 방송중인 홈쇼핑 채널을 볼 수 있습니다.
방송을 시청하면서 물건을 구매하고 결제할 수 있습니다.(카카오페이를 지원합니다.)
